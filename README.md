# 💳 Credit Default Risk Prediction

## 📌 Problem Statement
Predict whether a customer will default on credit card payment next month.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## 📊 Dataset
- Credit Card Default Dataset (30,000 records)
- Features include payment history, bill amounts, demographics

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

LINK : https://763wpz2dc7lbkapptpxluhr.streamlit.app/

## ⚙️ ML Pipeline
- Data Cleaning & EDA
- Handling Multicollinearity (VIF)
- Outlier Treatment
- Feature Engineering
- Encoding & Scaling (ColumnTransformer)
- SMOTE for imbalance
- Model Training with Cross Validation
- Hyperparameter Tuning (GridSearchCV)

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


## 🤖 Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- XGBoost ⭐
- SVM
- KNN
- Naive Bayes

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


## 📈 Evaluation Metric
- F1 Score (due to class imbalance)

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


## 🏆 Best Model
- XGBoost achieved highest performance
